# Requirements
In Visual Studio:

  1. Solution explorer
  2. Python enviroments
  3. Right click, view all enviroments
  4. Click overview and select (PyPI)
  5. Search for opencv (last update) I selected opencv-contrib-python (4.5.1.48)
  6. Now you are done

## OTHER REQUIREMENTS 
- Python 3.6
- `pip install opencv-contrib-python`
